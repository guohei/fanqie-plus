# Fanqie Export

Use the `fanqie-plus` skill.

Export final chapters to Fanqie-ready plain text:

1. Read `references/export-fanqie.md`.
2. Run or emulate `scripts/export_fanqie.py`; keep its default mechanical gate enabled.
3. Verify exported files contain正文 only, with no Markdown, metadata, indentation, or blank-line gaps.
4. Use `--no-gate` only for diagnostic cleanup, not for publish-ready export.
