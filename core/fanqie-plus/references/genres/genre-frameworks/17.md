## 核心梗解析与运用

**三代核心梗理论**

三代核心梗不是分类，而是**迭代层次**——每一层建立在上一层之上，一本小说的核心通常由 2-4 条核心梗融合而成。

**一代：主题/中心思想**

| 维度 | 说明 |
|------|------|
| 定义 | 小说的主题和中心思想 |
| 作用 | 赋予目标和立意，防止小说偏离核心 |
| 关键 | 没有主题 = 立意不明确 = 写出来的文字干巴巴没有情绪 |

主题在一本书的不同阶段可能有不同表现，不一定只有一个主题。

**二代：题材核心**

| 维度 | 说明 |
|------|------|
| 定义 | 该题材独有的核心吸引力 |
| 作用 | 筛选读者、划定边界、产生氛围感 |
| 提取法 | 看同题材火书的核心卖点和核心情绪 |
| 取巧法 | 题材名称本身即为线索（都市恋爱 = 在都市中谈恋爱） |

**核心原则：偏离题材核心 = 抛弃吸引读者的最大依仗，在读直线下降。**

即使戏剧性还在、其他核心梗还在运转，只要独属于该题材的核心梗偏离了，读者就会转头就走。

题材核心偏离案例：

| 作品 | 题材 | 偏离表现 | 后果 |
|------|------|----------|------|
| 《青山》 | 玄幻+悬疑 | 悬疑核心写跑偏了，以悬疑写玄幻 | 越写越乱，走下坡路 |
| 《一级一个金词条》 | 都市高武 | 对比对象从地球人跳到异世界神，超凡对比消失 | 在读直线下降 |
| 《普攻永久加生命》 | 网游文 | 走出新手村后仍和 NPC 打，偏离玩家互动核心 | 热度持续下滑 |
| 《天倾之后》 | 玄幻热血爽文 | 掺入上一本书的诡异恐怖设定 | 核心情绪断裂，两边不讨好 |

**三代：核心情绪**

| 维度 | 说明 |
|------|------|
| 定义 | 一种核心情绪体验（扮猪吃虎、弥补遗憾、改变命运、莫欺少年穷等） |
| 作用 | 筛选读者、提升戏剧性 |
| 特点 | 三代核心梗是展现读者需求与情绪的最佳形式，戏剧性最强 |

核心情绪不等于爽点表现形式。装逼打脸只是爽点的表现形式，不是核心梗本身。核心梗是完整的情绪链条（期待 -> 满足），辅以更具体的核心情绪来充实导向。

**剧情构建顺序**

**核心卖点 -> 情绪套路 -> 具体剧情**

先确定核心情绪（读者要看什么），再设计体现核心的情绪套路（怎么让读者感受到），最后根据套路设计具体剧情（用什么事件）。

**表达欲偏离核心梗的警示**

作者表达欲旺盛时容易偏离核心梗和读者需求。判断方法：写完一段剧情后站在读者角度审视，是否偏离核心梗？常见表现：

1. 对核心梗认知不够，判断失误
2. 认知够了但思维没切换到读者角度

**读者需求本质**：需求的内核从古至今不变，改变的永远是形式。理解这一点就能精准把握读者需求——不管载体怎么变，读者要的情绪体验是一致的。

**偏离核心梗的典型案例**

| 作品 | 偏离方式 | 后果 | 改善方向 |
|------|----------|------|----------|
| 《完美世界》 | 后期核心情绪崩塌，主角独断万古却啥也得不到 | 月票榜直线下降 | 独断万古可以，但主角要有实质收获 |
| 《十方武圣》 | 天地独行卷杀光所有人，换地图重建 | 核心情绪严重崩坏，订阅腰斩 | 换地图可以，不要杀光旧角色 |
| 《诡舍》 | 所有角色都成了工具人，身边人全死 | 高开低走 | 让主角身边的人遭遇危机后最终克服 |
| 《一人镇守孤城》 | 始终沉浸悲壮，无情绪节奏变化 | 读者麻木，昙花一现 | 穿插其他情绪，不要只有悲凉 |

**关键原则**：
- 悲情悲剧不需通过身边人死绝来体现，多创造挫折即可
- 刀人不是目的，是引起更大情绪波动的手段，极其危险
- 主角的目标和收获必须让读者觉得有意义

**悲剧命运改写法（有品质剧情设计）**

从《从斩妖除魔开始长生不死》提炼出的剧情设计方法：

1. 选取一个**悲剧原型**（纨绔强暴穷苦少女 / 私奔少女被捉回家折磨至死）
2. 选取一个**切入点**（主角介入的时间节点）
3. 选取一个**装逼方式**
4. 敲定一个贯穿故事表现力的**道具**（木棍、刀、弓）
5. 选取参与事件的**配角**（一般不超过 5 人）
6. 进行事件的**重塑**，构想装逼完成改写命运的画面
7. 处置好**悲剧变喜剧结局**的反转、主角的收获、下一个故事的引子

核心逻辑：把行侠仗义精神具体化，把俗的打怪装逼高尚化，把改写他人命运的命题小白化——融为一体。

### 核心梗的定义

- 核心梗 = 金手指最核心的使用方式 + 读者最想看的爽点模式
- 金手指是工具，核心梗是这个工具最有趣的使用方式
- 例：贡献毒计（金手指运用方式）+ 震惊女帝/老朱（爽点模式）= 核心梗
- 核心梗可以平替：女帝→朱元璋→嬴政，背景任意置换

### 核心梗的每章循环检查

每章必备要素（三选一）：
1. 拥有核心梗相关的期待点
2. 拥有核心梗相关的爽点
3. 介于核心梗的期待点和对应爽点之间

**断期待的判断**：某一章既没有核心梗的期待点也没有爽点，前一个爽点已结束，本身又没有拉起新期待感 → 读者觉得书失去了"灵魂"，开始弃书。

**两种运转模式**：
- 不断循环型：每个一级结构运行一次，变奏的是场景、对象、条件
- 只运行一次型：围绕一个核心情绪分阶段推进。例：禁忌之恋→"自我认知→告白→发糖→隐藏→摊牌"五阶段

### 核心梗的检验清单

- 读者翻开这本书最想看什么？（核心梗）
- 每个剧情单元是否都在展示核心梗？
- 核心梗的爽点模式能否循环使用？
- 有没有挂羊头卖狗肉（书名暗示的核心梗与实际内容不符）？
- 全文每一章是否满足三要素（期待点/爽点/中间态）中的至少一个？

### 拆书分析核心梗

- 先明确该书的一代（主题）、二代（题材核心）、三代（核心情绪）
- 分析该书的期待感和爽感分布，区分与核心梗相关和不相关的
- 白金大神的书通常有2-4条核心梗融合，需逐层拆解

### 核心梗常见误区

| 误区 | 表现 |
|------|------|
| 核心梗太多 | 什么都想写 = 没有核心梗 |
| 立意不明确 | 连情绪目标都没有，文字干巴巴 |
| 纯装逼打脸无主题 | 剧情本身没有意义 |
| 核心梗用完不升级 | 生硬重复，读者审美疲劳 |
| 挂羊头卖狗肉 | 书名暗示的核心梗与实际内容不符 |
| 认知偏差 | 不理解理论时先用本能写"我觉得爽"，写完再拿理论印证 |
