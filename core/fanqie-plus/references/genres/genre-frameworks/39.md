## 书名命名技法

### 身份法（覆盖市面上约1/3书名）

**第一类：高逼格身份**
- 前半句放绝对至高的身份，后半句写爽点变种
- 公式：`[仙帝/魔神/财阀]爹妈 + [反差行为/目标]`
- 范例："武神爹，财阀妈，我当躺赢狗怎么了？"

**第二类：戏剧身份对照**
- 找一对有差距且有关联的身份，做位置置换
- 公式：`[低位身份]替[高位身份]做[超出预期的事]`
- 步骤：找身份对（替身vs鲜肉）→ 找共同目标（要红）→ 置换结果
- 范例："让你当书童，你替少爷科举中状元？"
- 进阶变种：删除高位身份，只留低位身份+超出预期的结果
  - "让你做武打替身，你怎么成功夫巨星了？"

**第三类：身份擦边型**
- 用身份关系制造暧昧暗示，有风险但吸量极强
- 范例："前女友母亲怀上双胞胎"

### 其他书名方法概览

| 方法 | 核心 | 适用平台 |
|------|------|----------|
| 掉马甲型 | 隐藏身份逐渐暴露 | 已过度使用，慎用 |
| 伪同人类 | 原创中加入同人热词元素（甲斗腰带/八奇技） | 番茄/飞卢 |
| 逼格起名法 | 用意境感强的词汇 | 起点 |
| 反义词法 | 正反反差制造好奇 | 通用 |
| 金手指法 | 直接把金手指写在书名里 | 番茄/飞卢 |
| 主线+爽点法 | 主线剧情核心+一个爽点暗示 | 起点 |
| 乐子文法 | 一句话就让人想笑 | 番茄/飞卢 |

### 平台起名差异

| 平台 | 起名风格 | 关键原则 |
|------|----------|----------|
| 番茄 | 噱头优先，吸量第一 | 书名决定80%流量上限，相当于免费抖加 |
| 起点 | 轻度反差+金手指+主线+爽点 | 过度噱头会榨干前期潜力，内容必须贴合书名 |
| 飞卢 | 极致吸量，全网最会起名 | 可以参考飞卢榜单学习起名技巧 |

### 番茄书测改名策略

百万字书测时的五个书名选取法：
1. 金手指相关的一个
2. 第一个单元剧情相关的一个
3. 主线相关的一个
4. 与原书名差一个字的保底款
5. 一个稍微夸张吸量的款
