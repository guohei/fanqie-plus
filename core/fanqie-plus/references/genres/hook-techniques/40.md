## 噱头狗哨理论

**噱头 = 狗哨 = 特定群体快速识别的关键词**：
- 狗哨效应：只有特定群体能 get 到的信号——如火影写轮眼/海贼恶魔果实/克苏鲁/规则怪谈
- 噱头的作用是快速筛选目标读者——看一眼书名就知道"这是我的菜"
- 没有噱头 = 没有信号 = 读者滑走，再好的内容也没用
- 噱头要精准匹配目标读者群体——用错噱头比没有噱头更糟

**黄金三章本质 = 定调 = 预期管理 = 回应期待**：
- 黄金三章不是写好前三章就完了，本质是向读者传递"这本书能给你什么"
- 定调 = 明确告诉读者这本书的爽点类型和节奏
- 预期管理 = 你建立的期待必须在后续章节中回应
- 不回应期待 = 失信 → 读者弃书 → 比没有期待更糟糕
