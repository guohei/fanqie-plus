## 驱动力公式

**核心公式**：产生诉求 → 给予希望 → 努力解决 → 得偿所愿

**产生诉求（制造不爽）**：
- 低地位/困境/迫在眉睫的威胁/不曾拥有的事物
- 读者看到诬告/不公/恃强凌弱 → 心中天生生出"不该如此"的念头
- 这种"不该如此"的冲动 = 最根本的驱动力
- 好的题材具备勾起人们惋惜、渴望改变的特质，不单是"有趣"

**给予希望（金手指的本质）**：
- 金手指的本质 = 现实中我们难以拥有的特质/能力
- 读者感受到困境后需要看到改变的希望 → 没有希望 → 困境的痛苦让读者看不下去
- 清晰展示金手指 = 给读者愿意看下去的动力

**努力解决与节奏**：
- 不能让读者太快得偿所愿 → 心气平复后看下去的动力就没了
- 方式一"逐渐变好"：困境拆分为多个层级 → 层层递进
- 方式二"新的困境"：解决一个困境 → 遇到新的困境 → 困境层级不断提升
- 好的节奏：困境层级层层提高，难度和复杂度逐步提升，种类不尽相同

**"悬而未决"技巧**：
- 持续的"未解决"状态 = 持续的关注和期待
- 设置信息差 → 读者知道但角色不知道 / 角色知道但读者不知道
