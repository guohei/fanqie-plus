## 装逼打脸章节节奏模板

**爽点公式**：爽点 =（人设 + 铺垫）+ 人物的碰撞交织

**前置小无敌原则**：
- 装逼场景开始前，必须把道具/人物/实力这些打脸用的东西都准备好
- 没有前置小无敌 → 读者为主角担惊受怕 → 你写嘲讽读者觉得是虐主
- 有了前置小无敌 → 读者轻松看主角装逼 → 嘲讽只是铺垫

**铺垫比打脸更重要**：
- 都市修仙 KTV 装逼：高潮打脸 = 千把字，铺垫 = 几万字
- 铺垫到位后，主角站在那里一言不发都可以装逼打脸

**章节节奏模板**：

| 阶段 | 公众期（5章循环） | VIP期 |
|------|------------------|-------|
| 人设 | 1章 | 1-2章 |
| 铺垫 | 2章 | 5章 |
| 打脸 | 1章 | 1-2章 |
| 善后 | 1章 | 1章 |

- 从节奏模板看出：铺垫才是整个装逼打脸最核心、最值得详细描写的部分
- 打脸只是千把字的收尾

**装逼的逼格管理**：
- 神豪不能和阿猫阿狗装逼 → 逼格不对等 = 尴尬
- 路人甲震惊不叫爽 → 读者要的是有分量的人震惊
- 打脸对象的层级决定爽感的层级
