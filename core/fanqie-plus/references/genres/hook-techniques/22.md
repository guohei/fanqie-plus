## 事业线剧情循环设计

事业线的核心在于**地位变化**和**人际关系变化**，升级只是手段不是目的。

### 标准升级流循环

```
小事件 → 解决 → 收获（金钱/材料/配方/道具/关系）
→ 通过交易/社交获得升级信息
→ 升级 → 面对更高层次的挑战 → 循环
→ 最终收束到大高潮
```

### 循环中的要点

- **收获不能只有等级**：金钱、装备、人脉、信息都算收获
- **升级信息来源多样化**：交易、社交、探索、任务奖励均可
- **每次循环引入至少一个新变量**：新角色、新势力、新规则
- **控制支线密度**：小事件太多、人物出场太多容易出现琐碎问题

### 都市高武可用情节库

| 阶段 | 可用事件 |
|------|----------|
| 学校 | 天才班、虚拟比赛、班级打兽潮、月考、校霸、联考、高考 |
| 武馆 | 获馆主赏识、传承武学、师兄师姐、武馆联赛（市→省→国） |
| 治安局 | 打劫匪、打邪魔、打副本、打城内 boss |
| 竞争 | 武馆合约、学校助学金、武道厅选拔、军部招募、虚拟实战网、明星赛事 |

> 所有目标都必须和"钱"挂钩，反复挂钩。没钱寸步难行，有钱打遍天下——这是小人物代入感的核心。
