## 三翻四震结构

一种递进式震惊写法，本质是"情绪升级 + 波折起伏"。

### 结构

1. 主角宣布/展示第一层 → 众人反应
2. 更深层揭示 → 众人反应升级
3. 高位者介入质疑 → 众人反应再升级
4. 终极揭示 + 权威背书 → 全场震撼

### 实例拆解（主角选择院校）

```
主角称已有心仪院校 → 众人反应
主角宣布选龙科大（非清北）→ 众人反应
清北教授不解追问 → 众人反应！
龙科大校长高度评价"国士无双"→ 众人反应！
落选专家垂头丧气离开（压一压）
```

### 关键原则

- 每一翻都要有新的信息量，不是重复同一信息
- 震惊的层级和范围要递进：个人 → 小群体 → 全场 → 权威人物
- 最后可以用一个"压一压"收尾（配角失落离开），避免情绪过于单一
