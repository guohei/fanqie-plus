## 拉期待的86种手法

以下手法按功能分为八大类，可自由组合使用。

### 角色行为类
| 手法 | 说明 |
|------|------|
| 深入人心 | 主角持续扮演某个身份/角色，读者期待真实身份暴露的时刻 |
| 惊世之举 | 主角运用特殊能力解决问题/装逼，读者期待更大规模的使用 |
| 反差行为 | 角色做出与外在形象完全不符的行为，制造惊喜 |
| 隐藏实力 | 主角展现部分实力，读者期待全面爆发 |

### 时局/事件类
| 手法 | 说明 |
|------|------|
| 波澜乍起 | 突发意外打破平静，引发新的局势 |
| 乘间伺隙 | 角色在暗中蓄力准备，读者期待亮剑时刻 |
| 火中取栗 | 在危险中谋取利益，紧张感拉满 |
| 浑水摸鱼 | 在混乱局势中巧妙周旋，读者期待主角的手段 |

### 人际关系网扩展类
| 手法 | 说明 |
|------|------|
| 千里缘牵 | 不同地图/势力的角色因各种原因产生联系 |
| 众说纷纭 | 多方势力对主角/事件的不同反应和讨论 |
| 声名远播 | 主角名声向外扩散，引起新势力的注意 |
| 春风化雨 | 支线角色因主角的帮助而境遇改变 |
| 金尽裘敝 | 重要角色遭遇麻烦，读者期待主角出手 |

### 目标/冲突/矛盾升级类
| 手法 | 说明 |
|------|------|
| 计划进度 | 主角逐步接近目标，每一步都拉起新期待 |
| 察觉征兆 | 暗中威胁的线索逐渐浮现 |
| 冲突必爆 | 不可调和的矛盾积累到爆发临界点 |
| 再惹麻烦 | 解决一个麻烦后引出更大的麻烦 |
| 升级反应 | 各方势力对主角升级/突破的反应 |

### 资源/实力提升类
| 手法 | 说明 |
|------|------|
| 珍稀资源 | 主角获得稀缺资源，读者期待其用途 |
| 变废为宝 | 主角用独特方式将看似无用的东西变成宝物 |
| 对症之药 | 主角恰好拥有解决当前困境的手段 |
| 判若云泥 | 主角掌握的信息/实力远超他人认知 |

### 新线索/设定类
| 手法 | 说明 |
|------|------|
| 新地图 | 即将前往的新区域/新世界 |
| 世界观 | 新的规则/设定/体系的揭示 |
| 功能组合 | 已有能力的新组合方式 |
| 别有洞天 | 隐藏在已知信息下的更深层内容 |
| 拨云见日 | 谜团逐渐揭开的过程 |
| 峰回路转 | 看似绝路的局面出现转机 |

### 实力等级/技能类
| 手法 | 说明 |
|------|------|
| 解之以级 | 描写升级路径，读者期待每次突破 |
| 解之以技 | 展示新能力的具体效果 |
| 等级优势 | 升级后期待装逼剧情出现 |
| 天然克制 | 主角能力恰好克制当前对手 |
| 层出不穷 | 晋升后展现不同新能力 |

### 情绪/氛围类
| 手法 | 说明 |
|------|------|
| 雾里看花 | 适当的谜语人手法，隐藏部分信息 |
| 死得其所 | 重要角色的牺牲带来的情绪冲击 |
| 运筹帷幄 | 主角在暗中布局，读者期待收网 |

### 组合使用原则
1. 技巧服务表达，不要为了用技巧而用技巧
2. 构思剧情阶段可同时考虑多种技巧丰富剧情
3. 技巧可杂糅使用，没用就丢弃
4. 不要通过单个技巧推导剧情，技巧是工具不是束缚
