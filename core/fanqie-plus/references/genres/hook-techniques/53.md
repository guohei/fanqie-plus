## 智斗副本写法与解密公式

**智斗副本五步法**：
1. **定规则**：先设定副本规则，规则本身制造限制和压力
2. **探索规则**：在探索中展现队友性格，同时凸显主角的洞察力和决策力
3. **拆分内外线**：把玩家内斗和副本解谜分两部分，先解决内斗再解决副本
4. **用选择划分阵营**：通过不同选择（是否下地铁/是否抱团/是否探索）筛选敌友
5. **解密收尾**：拼凑完整答案 → 验证 → 大战 boss

**解密公式**：
规定规则 → 探索规则获得信息和疑惑 → 交流情况获得额外信息 → 推测信息得到接近答案的真相 → 解开疑惑拼凑完整答案 → 验证规则大战

**主角凸显智力的写法**：
- 在讨论中发表三次关键意见，每句话都展示一个维度
- 假装吃人玩家钓鱼——制造信息差，让读者知道主角在下套

**日常写法公式**：
日常 = 人设 + 有目的的日常 + 信息
- 每段日常必须推进某个目的（测试能力互补/铺垫线索/交代设定）
- 人设是日常的核心
