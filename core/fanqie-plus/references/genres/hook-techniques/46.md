## 看点爽点双逻辑体系

**看点 = 吊起读者期待的点，爽点 = 满足读者期待的点**。一一对应 → 前面铺垫多少爽点就有多少。

**爽点核心公式**：
- 爽点 = 两个逻辑的冲突点
- **大众逻辑** = 常规认知 → 对普通人来说对手不可战胜 → 恐惧/绝望
- **主角逻辑** = 主角视角 → 正好手痒/轻描淡写
- 两个逻辑落差越大 → 看点越足 → 爽点越爽

**爽点三要素铁三角**：主角 + 对手 + 吃瓜

- 吃瓜绝对不是无脑喊 6 → 三要素中重点其实在吃瓜上
- 吃瓜作用①：拉高期待 → 用议论把对手行为吹出逼格 → 让爽点更爽
- 吃瓜作用②：完成装逼打脸 → 吃瓜的震惊反应本身就是爽点 → 同时完成爽点升华
- 吃瓜 = 无时无刻提醒读者大众逻辑的角色 → 找出每个反差点并进行反应

**网文精髓 = 建立反差 + 写反应**：
- 反差 = 看点 → 读者期待的不是反差本身 → 而是反差之后的反应 → 这个才是爽点
- 反应由吃瓜提供最多 → 所以吃瓜视角最重要
- 看点部分往死里铺垫 → 爽点部分要收 → 给甜头但不完全满足 → 读者保持欲求不满
