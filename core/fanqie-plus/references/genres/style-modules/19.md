## 毒点解析与规避

### 五大毒点方向

| 方向 | 说明 | 规避方法 |
|------|------|----------|
| 主角塑造 | 主角行为/三观/思考读者无法认同 | 作者屁股坐在主角那边；主角做事必有正反馈 |
| 题材雷点 | 不同题材读者群体雷点不同 | 了解目标读者群体，避开敏感话题 |
| 氛围代入感 | 突然蹦出来的烂梗破坏沉浸 | 氛围感比梗重要一百倍（乐子文除外） |
| 期待破坏 | 长时间不满足读者期待 | 把握节奏，适时满足或引入新期待 |
| 题材选择 | 盲目跟风大神题材 | 参考三五千均订的作品而非顶流 |

### 核心原则

- **主角是心头爱**：作者屁股必须在主角那边，否则等于拿读者的钱恶心读者
- **明确读点**：毒点随题材变化，讨好目标受众而非所有读者
- **期待管理**：连续多次破坏读者期待会引发信任危机
- **开头简洁**：用最少的字数展示卖点和受众，第一章就展现关键要素
- **公式化写作有效**：公式化作品确实有稳定读者群
