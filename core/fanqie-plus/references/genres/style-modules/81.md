## 文笔的本质与层级

### 写作能力三块板

- **选材**（市场嗅觉）：判断什么题材有市场、什么元素受读者欢迎
- **剧情设计**（大纲/卷纲/章纲）：搭建故事框架和节奏
- **文笔**（行文）：把脑中的故事落实到文字上

三者的关系：只会文笔 = 枪手，只会剧情 = 纲手，只会选材 = 跟风，三项合一 = 健全写手。

### 文笔重要性的阶段差异

文笔的重要性因写作阶段而异：

| 阶段 | 文笔重要性 | 说明 |
|------|-----------|------|
| 小学生文笔时 | 10分（极度重要） | 基础不过关，什么都谈不上 |
| 合格扑街时 | 5分（不太重要） | 文笔达标后，短板在结构和选材 |
| 精品级别 | 7分（比较重要） | 精品之间的竞争，文笔开始拉开差距 |
| 万订时 | 10分（极度重要） | 顶级作品的文笔是核心竞争力之一 |

**关键判断**：
- 成绩好但文笔差 → 提升文笔的性价比极高，投入产出比最大
- 成绩差但文笔好 → 提升文笔毫无意义，应该去补选材和剧情设计

### 文笔四要素

| 要素 | 说明 |
|------|------|
| **画面感** | 人 + 环境（静/动），如高清摄像机记录可观测的一切 |
| **思想** | 计划/目标/吐槽/揣测/分析等内心活动 |
| **情绪** | 藏在画面感与思想之中，不需要单独描写 |
| **旁白** | 叙述者的补充说明和评价 |

**画面感**是最基础的要素：先有画面，读者才能"看到"故事。画面感 = 人（外貌/表情/动作/姿态）+ 环境（静态背景 + 动态变化）。

### 情绪的特殊性

情绪最特殊：它不需要单独描写，而是藏在画面感与思想之中。读者从角色的表情、动作、内心活动中自己感受到情绪，比直接写"他很愤怒"有力得多。

### 表达情绪的层级（由低到高）

| 层级 | 写法 | 效果 |
|------|------|------|
| 最低 | 直接写"xx愤怒" | 告诉读者情绪，但读者感受不到 |
| 低 | 用思想"xx暗骂" | 通过内心活动暗示情绪，稍好 |
| 中 | 用神态 | "xx脸色铁青"，侧面展现 |
| 高 | 用动作 | "xx猛地攥紧拳头，指甲掐进肉里"，最生动 |

**核心原则**：离底层情绪越远的表达越高级 = 侧面描写。不写情绪本身，而是写情绪引发的外在表现，让读者自己去感受。
