## 新媒体写作与题材套路

### 新媒体文核心原则

**情绪第一**：新媒体的核心不是剧情，是情绪。所有技巧都为调动和释放情绪服务。

**情绪分类与对应关系**：
- 不爽 → 爽（通常通过装逼实现）
- 愤怒 → 解气（通过报复/惩罚实现）

情绪值分轻重两种：轻压 = 走路上撞人被骂；重压 = 女朋友被人捅了一刀。

### 三种装逼方式

**第一种：基础装逼**（不压，直接装）
- 利用人性的表现欲——你很牛逼但无人知道，你表现出来就是装逼
- 例：地震时闪电般速度救下众人，所有人震惊
- 飞卢文常用：突然装逼，打对方措手不及，路人惊叹

**第二种：温酒斩华雄式装逼**（侧面压，装逼）
- 基础装的升级版——用强烈对比烘托主角牛逼
- 配角突出反派的牛逼，反派突出主角的牛逼
- 压得不重，一切服务于更好的装逼

**第三种：狠压式装逼**（核心中的核心）
- 温酒斩华雄的再升级——在第二种基础上狠压，压到读者上头再装逼
- 例：关羽投靠曹操被打杂，所有人都贬低他，结果华雄一刀被斩
- 不是单纯的装逼，而是**打脸+装逼的终极爽**

### 三种打脸方式

**第一种：愤怒式打脸**（单层次）
- 别人瞧不起男主+做了令读者上火的事 → 打脸一次
- 例：拆穿反派假画

**第二种：循环打脸**（多层次递进）
- 同一剧情打三四次，一波比一波爽
- 例：拆穿假画 → 奶奶偏袒反派说是真画 → 假画上写着"中国制造"继续打脸

**第三种：狠压式打脸**（核心）
- 压得越狠，打脸越爽
- 例：买车被狗眼看人低 → 买下砸了 → 再买再砸 → 暴露身份，对方不敢收钱
- **愤怒式与瞧不起式是两种不同压点，千万不能写串**：愤怒式是做了令读者上火的事；瞧不起式是狗眼看人低
